LEGO NXT Tacho Output Device

Description
The NXT tacho output device is as an example the "L-MOTOR-DEG" motor.
This device has the same type and caracteristics as the EV3 large motor.
Connection type of the motor is defined as shown in tabel Detection Rules.


Type/Mode
"Type/Mode" data is read at startup from the "typedata.rcf" file (see Type 7).



Byte Code Support
opOUTPUT_POWER
opOUTPUT_START
opOUTPUT_STOP
opINPUT_DEVICE_READY_SI



Byte Code Example
Write LEGO NXT Tacho Output Device



Schematics

  • Q1 and Q2 controls the 2 tacho phases.
