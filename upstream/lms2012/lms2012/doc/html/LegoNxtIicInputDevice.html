LEGO NXT IIC Input Device

Description
The NXT IIC input device is as an example the "NXT-US-CM" sensor.
Connection type of the sensor is defined as shown in tabel Detection Rules.


Type/Mode
"Type/Mode" data is read at startup from the "typedata.rcf" file (see Type 5).



Byte Code Support
opINPUT_DEVICE_READY_SI



Byte Code Example
Read LEGO NXT IIC Input Device



Schematics

  • "DATA" and "CLK" is the IIC digital signals that communicates the data.
