Branch

opJR (OFFSET)

  • Branch unconditionally relative

  • Dispatch status unchanged

    (DATA32)OFFSET

  • Branch relative if FLAG is FALSE (zero)

  • Dispatch status unchanged

    (DATA8)FLAG
    (DATA32)OFFSET

  • Branch relative if FLAG is TRUE (non zero)

  • Dispatch status unchanged

    (DATA32)OFFSET

  • Branch relative if VALUE is NAN (not a number)
