LEGO EV3 Tacho Output Device

Description
The EV3 tacho output device is as an example the "M-MOTOR-DEG" motor.
Connection type of the motor is defined as shown in tabel Detection Rules.


Type/Mode
"Type/Mode" data is read at startup from the "typedata.rcf" file (see Type 8).



Byte Code Support
opOUTPUT_POWER
opOUTPUT_START
opOUTPUT_STOP
opINPUT_DEVICE_READY_SI



Byte Code Example
Write LEGO EV3 Tacho Output Device



Schematics

  • "R1" value defines the "Type" (see EV3 Tacho Input ID's).
  • T1 and T2 is the 2 tacho phases.
