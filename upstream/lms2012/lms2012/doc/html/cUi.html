    opUI_READ (CMD, ....)

  • User Interface read

  • Dispatch status can change to BUSYBREAK and FAILBREAK

    Parameters
    (DATA8)CMD - Specific command parameter


    • CMD = GET_STRING
      Get string from terminal
      • Parameters
        (DATA8)LENGTH - Maximal length of string returned
      • Returns
        (DATA8) DESTINATION - String variable or handle to string

    • CMD = KEY
      Get key from terminal (used for debugging)
      • Returns
        (DATA8) VALUE - Key value from lms_cmdin (0 = no key)

    • CMD = GET_ADDRESS
      Get address from terminal (used for debugging)
      • Returns
        (DATA32) VALUE - Address from lms_cmdin

    • CMD = GET_CODE
      Get code snippet from terminal (used for debugging)
      • Parameters
        (DATA32)LENGTH - Maximal code stream length
      • Returns
        (DATA32) *IMAGE - Address of image
      • Returns
        (DATA32) *GLOBAL - Address of global variables
      • Returns
        (DATA8) FLAG - Flag tells if image is ready to execute [1=ready]

    • CMD = GET_HW_VERS
      Get hardware version string
      • Parameters
        (DATA8)LENGTH - Maximal length of string returned (-1 = no check)
      • Returns
        (DATA8) DESTINATION - String variable or handle to string

    • CMD = GET_FW_VERS
      Get firmware version string
      • Parameters
        (DATA8)LENGTH - Maximal length of string returned (-1 = no check)
      • Returns
        (DATA8) DESTINATION - String variable or handle to string

    • CMD = GET_FW_BUILD

      • Parameters

      • Returns
        (DATA8) DESTINATION - String variable or handle to string

    • CMD = GET_OS_VERS

      • Parameters
        (DATA8)LENGTH - Maximal length of string returned (-1 = no check)
      • Returns
        (DATA8) DESTINATION - String variable or handle to string

    • CMD = GET_OS_BUILD

  • CMD = LED

  • CMD = SET_BUSY

  • CMD = POWER
    • Parameters
      (DATA8)VALUE - Value [0,1]

  • CMD = INIT_RUN
    Start the "Mindstorms" "run" screen


  • CMD = GRAPH_SAMPLE
    Update tick to scroll graph horizontally in memory when drawing graph in "scope" mode

    Send to brick when file down load is completed (plays sound and updates the UI browser)

  • CMD = SCREEN_BLOCK
    Set or clear screen block status (if screen blocked - all graphical screen action are disabled)

      (DATA8)STATUS - Value [0 = normal,1 = blocked]

  • CMD = TEXTBOX_APPEND
    Append line of text at the bottom of a text box

    • Parameters
      (DATA32)SIZE - Maximal text size (including zero termination)
    • Parameters
      (DATA8)DEL - Delimiter code
    • Parameters
      (DATA8)SOURCE - String variable or handle to string to append
